# 수능 빈출 문법 패턴 (Grammar Patterns)

수능 영어 어법 빈출 패턴을 정리합니다. (출제 비율: 동사 50%, 관계사 36%, 기타 14%)

---

## 1. 동사 관련 (50%)

### 1.1 주어-동사 수일치

**패턴**: 주어가 복수 → 동사도 복수

```
❌ The students in the class was excited.
✓  The students in the class were excited.
```

**구문분석**:
```
[The students (in the class)] were excited.
 ────────────  ────────────── ──── ────────
      S           전치사구       V     C
   (복수)         ↳ 함정!      (복수)
```

**함정 유형**:
| 함정 | 예시 | 핵심 주어 |
|------|------|----------|
| 전치사구 삽입 | The book *of poems* is... | book (단수) |
| 관계절 삽입 | The man *who works here* is... | man (단수) |
| 분사구 삽입 | Students *studying hard* are... | students (복수) |

---

### 1.2 동사 vs 준동사

**패턴**: 문장에는 반드시 동사(V)가 있어야 함

```
❌ The boy running in the park.
✓  The boy running in the park fell.
```

**구문분석**:
```
[The boy (running in the park)] fell.
 ───────  ────────────────────  ────
    S          현재분사구          V
              (수식어, 동사 ❌)   (진짜 동사)
```

**구분법**:
| 형태 | 동사 | 준동사 |
|------|------|--------|
| -ing | be동사 필요 (is running) | 단독 사용 (running boy) |
| p.p. | be/have 필요 (was broken) | 단독 사용 (broken window) |
| to-v | 단독 동사 ❌ | 명/형/부 역할 |

---

### 1.3 시제 일치

**패턴**: 주절 과거 → 종속절도 과거

```
❌ He said that he is tired.
✓  He said that he was tired.
```

**예외**: 불변의 진리, 현재도 유효한 사실
```
He said that the earth is round. ✓ (불변의 진리)
```

---

### 1.4 능동 vs 수동

**패턴**: 주어가 행위자 → 능동, 주어가 대상 → 수동

```
❌ The window broke by the ball.
✓  The window was broken by the ball.
```

**구문분석**:
```
The window was broken (by the ball).
─────────── ────────── ────────────
     S          V          M
  (대상)      (수동태)   (행위자)
```

**감정동사 주의**:
| 능동 (-ing) | 수동 (-ed) |
|------------|-----------|
| 사물 주어 | 사람 주어 |
| The movie is boring. | I am bored. |
| It is surprising. | I am surprised. |

---

## 2. 관계사 관련 (36%)

### 2.1 관계대명사 vs 관계부사

**핵심**: 뒤에 불완전한 문장 → 관계대명사, 완전한 문장 → 관계부사

```
✓ the place where I was born (완전)
✓ the place which I visited (불완전: 목적어 없음)
```

**구문분석**:
```
the place [where I was born]
           ───── ──────────
           관부   완전한 문장 (S+V)

the place [which I visited Ø]
           ───── ──────────
           관대   불완전 (목적어 없음)
```

**판별표**:
| 선행사 | 불완전 → | 완전 → |
|--------|----------|--------|
| 사람 | who/whom | - |
| 사물 | which | - |
| 장소 | which | where |
| 시간 | which | when |
| 이유 | which | why |

---

### 2.2 전치사 + 관계대명사

**패턴**: 전치사가 앞으로 이동

```
the book which I am interested in
= the book in which I am interested
```

**구문분석**:
```
the extent [to which scientists need to develop]
            ────────
           전치사+관대
           ↳ to the extent 에서 이동
```

**빈출 표현**:
| 전치사+관대 | 원래 형태 |
|------------|----------|
| in which | which ~ in |
| to which | which ~ to |
| by which | which ~ by |
| for which | which ~ for |

---

### 2.3 관계대명사 that vs what

**핵심**: that = 선행사 필요, what = 선행사 포함

```
❌ I know what book you bought.
✓  I know the book that you bought.
✓  I know what you bought. (= the thing that)
```

**구문분석**:
```
I know [what you bought].
       ──────────────────
       명사절 (목적어)
       what = the thing which
```

---

### 2.4 관계대명사의 계속적 용법

**패턴**: , which = and it/this

```
He passed the exam, which made his parents happy.
                    ──────
                    = and it (합격한 것)
```

**구문분석**:
```
He passed the exam, [which made his parents happy].
──────────────────   ───── ──────────────────────
      주절           관대      관계절(부가설명)
                    = and it
```

---

## 3. 준동사 관련

### 3.1 to부정사 vs 동명사

**동사별 목적어 결정**:

| to-v만 | v-ing만 | 둘 다 (의미 같음) | 둘 다 (의미 다름) |
|--------|---------|------------------|------------------|
| want, hope, decide, plan | enjoy, mind, avoid, finish | like, love, start, begin | remember, forget, try, stop |

**의미 차이**:
```
remember to-v = 앞으로 할 것을 기억하다
remember v-ing = 과거에 한 것을 기억하다

stop to-v = ~하기 위해 멈추다
stop v-ing = ~하는 것을 멈추다
```

---

### 3.2 분사 (현재분사 vs 과거분사)

**핵심**: 능동 → -ing, 수동 → p.p.

```
the singing bird (새가 노래함 → 능동)
the broken window (창문이 깨짐 → 수동)
```

**구문분석**:
```
(Seen from a distance), the mountain looks beautiful.
──────────────────────
     분사구문(수동)
     ↳ 주어(mountain)가 보여지는 것
```

**분사구문 해석**:
| 의미 | 접속사 대체 |
|------|------------|
| 시간 | when, while, as |
| 이유 | because, since, as |
| 조건 | if |
| 양보 | though, although |
| 동시동작 | and |

---

### 3.3 사역동사 + 목적어 + 원형/분사

**패턴**:
```
make/have/let + O + 원형동사 (능동)
make/have + O + p.p. (수동)
```

**구문분석**:
```
I had my car repaired.
─ ─── ────── ────────
S  V    O       OC
         ↳ 차가 수리당함 (수동)
```

---

## 4. 병렬 구조

**패턴**: and/or/but으로 연결된 요소는 문법적으로 동등

```
❌ I like swimming, hiking, and to run.
✓  I like swimming, hiking, and running.
```

**구문분석**:
```
I like [swimming], [hiking], and [running].
       ─────────   ────────      ────────
        동명사      동명사        동명사
            ↳ 모두 같은 형태 (병렬)
```

---

## 5. 비교 구조

### 5.1 원급/비교급/최상급

```
as + 원급 + as
비교급 + than
the + 최상급 + in/of
```

### 5.2 비교급 강조

```
much/even/far/still/a lot + 비교급
```

**주의**: very는 비교급 강조 불가
```
❌ very better
✓  much better
```

---

## 6. 대명사

### 6.1 지시대명사 that/those

**패턴**: 앞에 나온 명사의 반복 피하기

```
The climate of Korea is milder than that of Canada.
                                   ────
                                   = the climate
```

### 6.2 재귀대명사

**패턴**: 주어와 목적어가 같을 때

```
She hurt herself.
    ──── ────────
     S      O
     ↳ 같은 대상
```

## 빠른 체크리스트

어법 문제 풀이 순서:

1. **동사 확인**: 주어와 수일치? 시제? 태?
2. **관계사 확인**: 뒤 문장 완전/불완전?
3. **준동사 확인**: 문장에 동사 있음? 능동/수동?
4. **병렬 확인**: and/or 앞뒤 형태 동일?
5. **대명사 확인**: 지시 대상 명확?
