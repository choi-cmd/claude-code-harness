# 출력 템플릿 (Output Templates)

구문분석 결과를 출력하는 세 가지 형식입니다.

---

## 1. 기본형 (Default)

일반적인 구문분석 시 사용. 구조도 + 성분 분석표.

### 템플릿

```markdown
## 구문분석

### 원문
> {원문 문장}

### 구조도
\`\`\`
{구조도}
\`\`\`

### 성분 분석

| 기호 | 원문 | 성분 | 품사/기능 | 비고 |
|:----:|------|:----:|----------|------|
| {기호} | {원문} | {성분} | {품사} | {비고} |
```

### 예시

```markdown
## 구문분석

### 원문
> The book which I bought yesterday is interesting.

### 구조도
```
[The book (which I bought yesterday)] is interesting.
 ────────  ───────────────────────── ── ───────────
    S            관계절[형용사절]      V      C
                     ↳ book 수식
```

### 성분 분석

| 기호 | 원문 | 성분 | 품사/기능 | 비고 |
|:----:|------|:----:|----------|------|
| S | The book | 주어 | 명사구 | 핵심 주어 |
| [ ] | which I bought yesterday | 수식어 | 관계절 | book 수식 |
| V | is | 동사 | be동사 | 2형식 |
| C | interesting | 보어 | 형용사 | 주격보어 |
```

---

## 2. 상세형 (Detailed)

교재 해설용. 구조도 + 성분표 + 문법 포인트 + 해설.

### 템플릿

```markdown
## 구문분석 (상세)

### 원문
> {원문 문장}

### 구조도
\`\`\`
{구조도}
\`\`\`

### 성분 분석

| 기호 | 원문 | 성분 | 품사/기능 | 비고 |
|:----:|------|:----:|----------|------|
| {기호} | {원문} | {성분} | {품사} | {비고} |

### 문법 포인트

| 포인트 | 설명 |
|--------|------|
| {문법항목} | {설명} |

### 문장 형식
- **형식**: {N형식}
- **구조**: {S + V + ...}

### 해설
> {상세 해설}

### 어휘
| 단어 | 품사 | 의미 |
|------|------|------|
| {단어} | {품사} | {의미} |
```

### 예시

```markdown
## 구문분석 (상세)

### 원문
> The book which I bought yesterday is interesting.

### 구조도
```
[The book (which I bought yesterday)] is interesting.
 ────────  ───────────────────────── ── ───────────
    S            관계절[형용사절]      V      C
                     ↳ book 수식
```

### 성분 분석

| 기호 | 원문 | 성분 | 품사/기능 | 비고 |
|:----:|------|:----:|----------|------|
| S | The book | 주어 | 명사구 | 핵심 주어 |
| [ ] | which I bought yesterday | 수식어 | 관계절 | book 수식 |
| V | is | 동사 | be동사 | 2형식 |
| C | interesting | 보어 | 형용사 | 주격보어 |

### 문법 포인트

| 포인트 | 설명 |
|--------|------|
| **관계대명사** | which = 목적격 관계대명사, 선행사: book |
| **목적격 관계대명사** | bought의 목적어 역할, 생략 가능 |
| **문장 형식** | 2형식 (S + V + C) |
| **관계절 기능** | 형용사절로 선행사 book을 수식 |

### 문장 형식
- **형식**: 2형식
- **구조**: S(주어) + V(동사) + C(보어)

### 해설
> **which I bought yesterday**는 목적격 관계대명사 which가 이끄는 형용사절이다.
> 이 절은 선행사 the book을 뒤에서 수식한다.
> which는 관계절 내에서 bought의 목적어 역할을 하며,
> 목적격 관계대명사이므로 생략이 가능하다.
>
> 전체 문장은 2형식(S+V+C) 구조로,
> 주어부(The book which I bought yesterday)가 be동사 is를 통해
> 보어 interesting과 연결된다.

### 어휘
| 단어 | 품사 | 의미 |
|------|------|------|
| interesting | adj. | 흥미로운, 재미있는 |
| buy-bought-bought | v. | 사다, 구입하다 |
```

## 3. 간략형 (Simple)

빠른 확인용. 구조도만 출력.

### 템플릿

```markdown
## 구조도

\`\`\`
{구조도}
\`\`\`

**형식**: {N형식} | **핵심**: {핵심 문법}
```

### 예시

```markdown
## 구조도

```
[The book (which I bought yesterday)] is interesting.
 ────────  ───────────────────────── ── ───────────
    S              관계절              V      C
```

**형식**: 2형식 | **핵심**: 목적격 관계대명사 which
```

## 4. 어법 문제 분석형 (Exam Analysis)

수능/모의고사 어법 문제 해설용.

### 템플릿

```markdown
## 어법 문제 분석

### 원문
> {원문 문장} (밑줄/번호 포함)

### 구조도
\`\`\`
{구조도}
\`\`\`

### 선택지 분석

| 번호 | 원문 | 판정 | 분석 |
|:----:|------|:----:|------|
| ① | {선택지1} | ✓/❌ | {분석} |
| ② | {선택지2} | ✓/❌ | {분석} |
| ③ | {선택지3} | ✓/❌ | {분석} |
| ④ | {선택지4} | ✓/❌ | {분석} |
| ⑤ | {선택지5} | ✓/❌ | {분석} |

### 정답
**{번호}번** - {정답 근거}

### 오답 해설
- ①: {오답 이유}
- ②: {오답 이유}
...

### 출제 의도
- **유형**: {출제 유형}
- **함정**: {함정 요소}
- **팁**: {풀이 팁}
```

### 예시

```markdown
## 어법 문제 분석

### 원문
> Patients ①recovering from a brain injury ②affecting the motor system ③knows that simple tasks can be extremely difficult. The ④knowledge that we unconsciously have the ability ⑤to see what we are reaching for and adjust our movement accordingly is indeed surprising.

### 구조도
```
[Patients (recovering from a brain injury) (affecting the motor system)] ③knows
 ────────  ───────────────────────────────  ──────────────────────────   ─────
    S                  현재분사①                      현재분사②              V
  (복수)                ↳ patients 수식                 ↳ injury 수식       (단수❌)
```

### 선택지 분석

| 번호 | 원문 | 판정 | 분석 |
|:----:|------|:----:|------|
| ① | recovering | ✓ | 현재분사, patients 수식 (능동) |
| ② | affecting | ✓ | 현재분사, injury 수식 (능동) |
| ③ | knows | ❌ | 동사, 주어 patients(복수)와 수일치 오류 |
| ④ | knowledge | ✓ | 명사, 주어 역할 |
| ⑤ | to see | ✓ | to부정사, ability 수식 (형용사적) |

### 정답
**③번** - knows → know (주어-동사 수일치)

### 오답 해설
- ①: patients가 회복하는 것(능동)이므로 현재분사 recovering 적절
- ②: injury가 영향을 미치는 것(능동)이므로 현재분사 affecting 적절
- ④: 명사 knowledge가 주어로 쓰임, 적절
- ⑤: ability to-v (∼하는 능력), to부정사의 형용사적 용법

### 출제 의도
- **유형**: 주어-동사 수일치
- **함정**: 긴 수식어구(분사구 2개)가 주어와 동사 사이에 삽입
- **팁**: 수식어구를 괄호로 묶고 핵심 주어를 먼저 찾을 것
```

## 5. 복합문 분석형 (Complex Sentence)

여러 절이 포함된 복잡한 문장 분석용.

### 템플릿

```markdown
## 복합문 분석

### 원문
> {원문}

### 절 구조
\`\`\`
[주절] + [종속절1] + [종속절2] ...
\`\`\`

### 절별 분석

#### 주절 (Main Clause)
\`\`\`
{주절 구조도}
\`\`\`
- **형식**: {N형식}
- **구조**: {S+V+...}

#### 종속절1: {절 종류}
\`\`\`
{종속절 구조도}
\`\`\`
- **접속사**: {접속사}
- **기능**: {명사절/형용사절/부사절}

### 전체 해설
> {해설}
```

## 출력 형식 선택 가이드

| 상황 | 권장 형식 |
|------|----------|
| 빠른 확인 | 간략형 |
| 일반 학습 | 기본형 |
| 교재 해설 작성 | 상세형 |
| 수능/모의고사 해설 | 어법 문제 분석형 |
| 긴 복합문 | 복합문 분석형 |

### 형식 지정 방법

```
"문장" 구문분석 해줘           # 기본형
"문장" 구문분석 해줘 --detailed # 상세형
"문장" 구문분석 해줘 --simple   # 간략형
"문장" 어법 분석 해줘           # 어법 문제 분석형
```
