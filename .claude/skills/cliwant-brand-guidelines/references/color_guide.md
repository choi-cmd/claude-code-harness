# Cliwant Color Reference

## Primary Brand Colors

### CLIWANT Purple
- **HEX**: #686FE8
- **RGB**: 104, 111, 232
- **CMYK**: 55, 52, 0, 9
- **Usage**: Primary brand color, main identifier

### CLIWANT Dark Blue
- **HEX**: #001376
- **RGB**: 0, 19, 118
- **CMYK**: 100, 84, 0, 54
- **Usage**: Secondary brand color, logo dark elements

### CLIWANT White
- **HEX**: #FFFFFF
- **RGB**: 255, 255, 255
- **CMYK**: 0, 0, 0, 0
- **Usage**: Logo version for dark backgrounds

## Background-Based Logo Color Selection

### White/Light Backgrounds (Brightness 0-30%)
**Logo Color**: Standard (Purple + Dark Blue)
- This is the 1st priority usage
- Use on pure white or very light backgrounds
- Most preferred application

### Medium Backgrounds (Brightness 30-50%)
**Logo Color**: Use with caution
- May need adjustment
- Test visibility before use

### Medium-Bright Backgrounds (Brightness 50-60%)
**Logo Color**: Standard (Purple + Dark Blue) or White depending on exact brightness
- Test for optimal visibility

### Bright Backgrounds (Brightness 60%+)
**Logo Color**: White
- For light-colored backgrounds
- When background is needed, use white logo with brand color vertical lines

### Dark Backgrounds (Below 30% brightness)
**Logo Color**: Standard (Purple + Dark Blue)
- Works well on dark colors
- Maintains brand visibility

### Complex or Image Backgrounds
**Logo Color**: White
- Use white logo for complex imagery
- Position on lighter areas when possible
- Ensure sufficient contrast

### Similar Color Backgrounds
**Logo Color**: White logo with alternate brand color for icon lines
- When background is similar to CLIWANT Purple, use white logo
- Icon vertical lines can use alternate brand color
- Maintains differentiation

## Color Application Best Practices

1. **Always use exact color values** - Do not approximate
2. **Test on actual media** - Colors may appear different on screen vs. print
3. **Maintain contrast** - Ensure logo is always clearly visible
4. **Consider context** - Choose appropriate color version for each situation
5. **Preserve brand integrity** - Follow priority order when possible
